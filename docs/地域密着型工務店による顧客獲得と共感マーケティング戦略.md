# 地域密着型工務店による顧客獲得と共感マーケティング戦略

> この文書は、地域密着型工務店である正田工建がどのように地域住民に認知され、顧客との信頼関係を築き、集客に繋げているかという具体的なマーケティング手法とノウハウをまとめたものです。

- **対象読者**: 地域密着型ビジネスの経営者、マーケティング担当者、営業担当者
- **原本**: [Drive で開く](https://drive.google.com/file/d/1DnckZUMMdAyp1hLHcEEM_tiIu_IzY3dM/view?usp=drivesdk)
- **最終更新**: 2026-06-25
- **カテゴリ**: sop

## 要点

- ポスティングによる地域への継続的な情報提供と自己紹介
- 地域住民を対象としたイベント（感謝祭、木工教室など）開催による関係構築
- 施工現場を活用した集客と丁寧な仕事ぶりの訴求
- 共感マーケティング（FUVSの法則、ザイアンスの法則）による信頼度向上
- 共感通信（ストーリーチラシ）の作成と活用
