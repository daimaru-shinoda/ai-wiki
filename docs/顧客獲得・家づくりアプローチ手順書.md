# 顧客獲得・家づくりアプローチ手順書

> 本手順書は、見込み客の獲得から成約に至るまでの顧客アプローチプロセスを詳細に説明し、自社の強みを活かした効果的な営業戦略を示すものです。

- **対象読者**: 営業担当者、マーケティング担当者、全従業員
- **原本**: [Drive で開く](https://drive.google.com/file/d/1_y18uKKJ5PDvU7sCka1PffF-Zo-7Zpsc/view?usp=drivesdk)
- **最終更新**: 2026-06-25
- **カテゴリ**: sop

## 要点

- 自社の強みである収納と家事動線を前面に押し出し、顧客の共感を呼ぶ。
- HP、イベント、チラシ、SNSなど多角的なツールを活用し、認知度向上と来社を促す。
- 顧客のアプローチを複数回に分け、信頼構築と不安解消を段階的に行う。
- 個別資金計画やOB訪問を通じて、具体的な費用と住み心地を伝える。
- デザイン、性能、収納などの自社優位性を明確に伝え、契約へ繋げる。
